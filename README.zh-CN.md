# Cappuccino

[English](README.md)

一款轻量级 WordPress 插件，可在文章底部自动插入赞赏图片（如微信或支付宝收款码）。

## 主要功能

- 通过媒体库上传赞赏图片
- 可配置提示文字，支持多行显示
- 可单独开关提示文字显示
- 可选点击图片跳转链接
- 后台勾选需要显示的文章类型
- 支持左对齐、居中、右对齐
- 可调节图片最大宽度

## 环境要求

- WordPress 5.8+
- PHP 7.4+

## 安装

1. 下载或克隆本仓库。
2. 将 `cappuccino` 文件夹复制到 `wp-content/plugins/` 目录。
3. 在 WordPress 后台「插件」页面激活 **Cappuccino**。
4. 进入「设置 → Cappuccino」上传图片并配置选项。

## 配置说明

| 选项 | 说明 |
| --- | --- |
| 启用赞赏 | 在所选文章类型底部显示或隐藏赞赏区块 |
| 赞赏图片 | 从媒体库选择收款码或赞赏图片 |
| 提示文字 | 显示在图片上方的说明文字，每行单独成段 |
| 点击链接 | 点击图片时跳转的链接，可留空 |
| 显示位置 | 勾选需要显示赞赏图片的公开文章类型 |
| 对齐方式 | 左对齐、居中或右对齐 |
| 图片最大宽度 | 图片宽度，单位 px（50–800） |

## 常见问题

**图片会显示在列表页吗？**

不会。插件只在单篇文章详情页（singular）底部显示。

**提示文字可以写多行吗？**

可以。在设置页的「提示文字」文本框中，每行会单独显示为一段说明。

**从 wp-donate 升级后设置会保留吗？**

会。Cappuccino 在首次加载时会自动从旧版选项键（`wpd_settings`、`wpad_settings`）迁移设置。
